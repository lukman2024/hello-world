Apa itu css?

SCSS adalah singkatan dari Sassy Cascading Style Sheets, sebuah varian CSS yang lebih maju1. SCSS adalah jenis file khusus untuk SASS, sebuah program yang mengolah data menggunakan syntax tertentu dengan output CSS23. SCSS menawarkan banyak fitur canggih seperti variabel, bersarang, dan lainnya yang dapat membuat penulisan CSS lebih efisien dan lebih
